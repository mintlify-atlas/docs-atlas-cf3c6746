Dashi is an AI-powered desktop assistant that combines voice input and chat capabilities in a sleek, animated interface. Built with modern web technologies and packaged as a native desktop application, Dashi provides seamless interaction with AI through two primary modes: voice conversations and text-based chat.

## Architecture overview

Dashi combines three main technologies:

- **Tauri**: Provides the desktop application framework and bridges the JavaScript frontend with Rust backend
- **Next.js**: Powers the React-based user interface with modern web technologies
- **AI Integration**: Uses Groq API for speech-to-text and Letta AI agents for intelligent responses

The Rust backend handles audio processing and API calls to Groq, while the React frontend manages the user interface, animations, and state management. This hybrid architecture leverages the strengths of both web and native development.
