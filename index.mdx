# Meet Dashi

Your AI-powered desktop assistant with voice input and real-time chat capabilities.

Dashi is a built with Tauri, Next.js, and Groq API to provide you with an intelligent assistant that responds to both your voice and text conversations.

## Quick Start

Get started with Dashi in minutes. Press the keyboard shortcuts to interact:

- `Cmd + Option + V` - Activate voice mode and start recording
- `Cmd + Option + T` - Switch to chat mode
- `Escape` - Reset to default state

## How It Works

Dashi combines modern web technologies with powerful AI services:

**Voice Recording** - Capture your voice directly in the browser using the MediaRecorder API

**Groq Transcription** - Convert speech to text using Groq's Whisper Large V3 model for fast, accurate transcription

**Letta Agents** - Generate intelligent responses through Letta's agent system with streaming output

**Desktop Integration** - Runs as a native desktop application packaged with Tauri for cross-platform support
